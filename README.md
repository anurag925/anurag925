I'm a passionate Software Engineer based in India who loves building developer tools, automation systems, and exploring modern runtime systems. I spend my time building fast, reliable backends in **Go**, crafting reactive frontends in **TypeScript/React**, and messing around with container infrastructures.

- 🚀 **Current Focus:** Creating developer experience (DX) tooling and task-runners.
- 🛠️ **Favorite Stack:** Go, Bun/Node.js, React, and Docker.
- 🧠 **Exploring:** LLM integrations, agent-based workflows, and systems-level programming.
- 🦈 **Fun Fact:** I'm an active open-source contributor (Proud holder of the Pull Shark x3 achievement!).
